# 闽台文化大辞典

> 闽台文化数字化阅读平台 —— 移动端单页应用

基于 React 18 + Tailwind CSS 构建的闽台文化辞典移动端阅读应用，支持简繁切换、语音朗读、智能搜索与 AI 问答。

## 功能特性

### 📖 辞典目录
- 六大卷目分类导航：地理卷、历史卷、民族与宗族卷、海洋文化卷、语言卷、民俗卷
- 彩色图标卡片依次动画进入
- 三级搜索匹配：精确匹配 → 包含匹配 → 关键词匹配

### 📑 阅读器
- 简体中文 / 繁体中文一键切换
- 字体大小调节（标准 / 大字版）
- 衬线字体正文排版（Noto Serif SC）
- 上一篇 / 下一篇词条导航
- 语义标签展示

### 🔊 语音朗读
- 普通话（zh-CN）/ 闽南语（zh-TW）双语音引擎
- 智能语音选择：精确匹配 → 前缀匹配 → 同语系匹配
- 闽南语语速自动降至 0.85x，贴合自然节奏
- 实时显示当前语音引擎名称

### 🤖 AI 问答
- 闽台文化知识库驱动的智能问答
- 5 大预设话题 + 自由提问
- 模糊匹配算法，支持关键词命中
- 模拟打字效果的流式回复

### 👤 个人中心
- 会员权益展示
- 已购纸书防伪码验证入口
- 阅读历史 / 我的订单 / 设置与帮助

## 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | React 18（UMD Production） |
| 样式 | Tailwind CSS（CDN） |
| 编译 | Babel Standalone（浏览器端 JSX） |
| 字体 | Noto Serif SC + Noto Sans SC（Google Fonts） |
| 语音 | Web Speech API（SpeechSynthesis） |
| 架构 | 单文件 HTML，零构建依赖 |

## 快速开始

### 本地运行

直接在浏览器中打开 `index.html` 即可，无需安装任何依赖：

```bash
# 方式一：直接打开
open index.html

# 方式二：本地服务器
npx serve .
```

### 在线访问

部署至任意静态托管即可使用：

- Cloudflare Pages
- Vercel / Netlify

## 项目结构

```
mintai-culture-dictionary/
├── index.html          # 完整单文件应用（HTML + CSS + JS）
├── README.md           # 项目说明
└── screenshots/        # 应用截图
```

所有代码集中在 `index.html` 一个文件中，包含：

- **CSS**：自定义样式 + Tailwind 工具类 + 动画定义
- **常量配置**：主题色、卷目分类、搜索结果、词条内容、知识库
- **子组件**：AudioPlayer、BottomNav、SearchResult、AIChat 等
- **主应用**：App 组件，管理页面路由与全局状态

## 自定义与扩展

### 修改主题色

编辑 `THEME` 常量：

```js
const THEME = {
  primary: '#B91C1C',      // 主色（中国红）
  primaryLight: '#fef2f2',
  accent: '#4f46e5',       // 强调色
  // ...
};
```

### 扩展知识库

向 `KNOWLEDGE_BASE` 对象添加新条目：

```js
const KNOWLEDGE_BASE = {
  '你的问题': {
    content: '回答内容',
    source: '来源卷目'
  },
  // ...
};
```

### 添加卷目分类

向 `CATEGORIES` 数组追加：

```js
const CATEGORIES = [
  { id: 7, title: '新卷目', sub: '描述', icon: '新', color: '#颜色' },
  // ...
];
```
